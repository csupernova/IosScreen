## Описание
<p>Данный проект был создан с помощью среды разрабоки XCode 13.3, ОС MacOS 12.6. </p>

## Технологии
- Язык программирования - Swift 5
- Набор инструментов для разработки пользовательского интерфейса - SwiftUI
